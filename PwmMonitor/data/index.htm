Welcome to openCross

MODE D'EMPLOI

Ce serveur local permet de piloter votre appareil via un navigateur web

Pour commencer :

  1. Désactivez les données mobiles
  2. Ouvrez votre navigateur web
    (e.g. chrome, safari, etc)
  3. Saisissez opencross.local dans la barre d'adresse

Si c'est déjà fait, cliquez sur Start !

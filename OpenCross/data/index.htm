OpenCross

Guide d'installation

1. Désactiver les données mobiles

2. Ouvrir votre navigateur web
 et allez à l'adresse suivante

 

3. Une fois que la page est ouverte,
 aller dans le menu du navigateur et
 cliquer sur "Ajouter à l'ecran d'accueil"

Les prochains démarrages se feront directement depuis votre écran d'accueil

Notes d'utilisation

OpenCross est une webapp sur serveur local. Son utilisation nécessite que le téléphone soit:

1. Connecté au réseau wifi OpenCross

2. Déconnecté d'internet
 (données mobiles désactivées)
